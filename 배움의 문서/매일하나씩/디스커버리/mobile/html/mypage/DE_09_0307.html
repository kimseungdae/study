공식 온라인 쇼핑몰에서 구매 후 적립되는 마일리지는 최대 5% 입니다.

Point
로그인 시 5P 적립

텍스트 리뷰 작성시 100P, 포토 리뷰 작성시 500P가 적립됩니다.

SNS 공유 시 5P 적립

상품리뷰 및 상품에 SNS 공유시 5P를 적립해드립니다.

혜택 사용 안내

  • 신규 회원 혜택 쿠폰/마일리지는 가입 이후 일정 기간 내 지급되며, 쿠폰/마일리지 사용 가능 기간에만 사용하실 수 있습니다.
  • 등급 쿠폰은 매월 8일에 전월까지의 구매이력으로 등급 산정하여 발급되며 발급 후 180일까지 사용 가능합니다.
    (승급 시 해당 쿠폰 발행)
  • 생일 쿠폰은 회원정보 상의 생일로부터 15일전 자동 발급되며 발급일로부터 45일까지 사용 가능합니다.
  • 신규 회원 쿠폰, 등급 쿠폰, 생일쿠폰은 온라인 공식몰, 오프라인 매장 중 한 곳만 선택하여 사용 가능합니다.
    (중복 사용 불가)
  • 마일리지는 5,000점 이상이 모이면 1,000점 단위로 사용 가능합니다.
  • 포인트는 1,000점 이상이 모이면 1,000점 단위로 사용 가능합니다.
